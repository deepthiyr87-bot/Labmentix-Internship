Dubai Real Estate Intelligence Dashboard

Objective:
An interactive and professional Power BI dashboard analyzing the Dubai housing market using property-level data. This project was built to uncover actionable insights for investors, real estate agents, and policymakers by exploring pricing patterns, property sizes, location trends, and market segmentation.

Project Objective
1) To design a high-level, interactive Power BI dashboard that helps users:
2) Understand average price trends across location types
3) Compare property size vs price visually
4) Analyze bedroom/bathroom configurations and property age distributions
5) Filter dynamically using slicers to drill into customized insights
